# Core Exam Module

## Implemented Runtime Boundaries
- Item bank: question authoring and metadata
- Paper studio: explicit composition
- Exam release: paper binding and candidate assignment
- Candidate center: save and submit
- Grading center: subjective grading
- Score center: published records
- Analytics: exam-level metrics
- Proctor view: anti-cheat events

## Package Ownership
- Backend controllers under `backend/src/main/java/com/projectexample/examsystem/controller`
- Backend services under `backend/src/main/java/com/projectexample/examsystem/service`
- Frontend exam views under `frontend/src/views/exam`
