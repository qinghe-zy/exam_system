# 试卷创建与组卷工作台说明

## 一、目标
将原来“大而杂的单弹窗建卷流程”拆为独立页面，降低教师在建卷时的认知负担。

## 二、当前页面结构
- 列表页：`/exam/papers`
  - 只展示试卷总览
  - 提供“新建试卷 / 新建随机组卷 / 新建策略组卷 / 编辑”入口
- 建卷页：`/exam/papers/create`
- 编辑页：`/exam/papers/:paperId/edit`

## 三、建卷页分区
### 1. 试卷基础信息
- 试卷编码
- 试卷名称
- 学科
- 组卷方式
- 试卷版本
- 考试时长
- 及格线
- 发布状态
- 卷面说明
- 版本备注
- 乱序开关

### 2. 组卷流程
- 手工选题：从题库筛选后加入卷面
- 随机组卷：按当前学科和题量目标直接抽题
- 策略组卷：按题型分布和难度分布自动拼装
- 卷面预览：统一查看题序、分值、题型和难度分布

## 四、当前校验
- 至少选择一道题目
- 试卷总分必须大于 0
- 及格线不能高于试卷总分
- 策略组卷时：
  - 题型分布合计必须大于 0
  - 题型分布与难度分布总题量必须一致
  - 当前题库必须能满足目标题型和难度组合

## 五、落库字段
- `biz_exam_paper.paper_version`
- `biz_exam_paper.remark_text`
- `biz_exam_paper.shuffle_enabled`
- `biz_exam_paper.question_type_config_json`
- `biz_exam_paper.difficulty_config_json`
- `biz_paper_question` 中的题序、分值、必答标记

## 六、验证结论
本轮已通过真实浏览器验证：
1. 手工选题建卷成功
2. 随机组卷建卷成功
3. 策略组卷建卷成功
4. 新建卷面可回到列表页并再次进入编辑
