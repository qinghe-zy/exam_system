# 测试文档说明

## 一、文档目的
本目录用于记录当前系统的构建验证、数据库回归、接口 smoke、浏览器级 E2E 和已知测试边界。

## 二、当前覆盖情况
当前测试覆盖已经不再局限于构建和接口 smoke，已包含：
- 后端测试
- 前端构建
- MySQL 实库核查
- Playwright 浏览器回归

当前已稳定通过的浏览器级 E2E 包括：
- 教师端建卷与发布链路
- 学生端进入考试、保存、提交链路
- 阅卷老师评分链路
- 权限路由拦截链路
- AI 题库辅助缺配置提示链路

## 三、当前限制
- 当前浏览器矩阵以 Edge 为主，尚未覆盖 Chrome / Firefox / Safari
- 当前并发压测已补基础登录压测脚本与说明，但尚未覆盖考试核心高并发链路
- 当前尚未覆盖弱网压测和移动端专项回归
