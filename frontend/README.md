# 前端说明

## 一、当前定位
当前前端负责在线考试系统的管理端与学生端页面，包括题库、试卷、考试发布、学生考试工作区、阅卷、成绩分析、消息中心和系统管理等视图。

## 二、当前实现重点
- 保留登录、布局、系统管理与首页看板基线
- 新增题库、试卷、考试发布、学生考试、阅卷、成绩分析、监考事件等业务页面
- 通过 Pinia 保存登录态和菜单树
- 通过 Vue Router + 后端菜单 + 前端路由守卫保持权限一致性

## 三、当前验证
- `npm.cmd run build`
- `npx.cmd playwright test`
- 已覆盖教师、学生、阅卷、权限路由与 AI 题库辅助的浏览器回归
