# 前端说明

## 定位

前端负责管理端与学生端界面，包括登录、首页看板、组织用户管理、题库、组卷、考试发布、学生考试工作区、阅卷、成绩分析、消息中心和系统配置。

## 当前重点

- 统一布局、顶部身份区和页面标题区
- 独立题目编辑页与独立组卷页
- 学生待考、考试工作区、成绩回看与错题本
- 基于后端菜单和权限的路由守卫

## 常用命令

```powershell
cd frontend
npm.cmd install
npm.cmd run build
npx.cmd playwright test
```

## 验证口径

- 生产构建通过
- 17 条 Playwright 浏览器回归全部通过
- 关键页面截图与交付图示已生成
